# Mẫu tài liệu (Tiếng Việt)

Trang này là cầu nối bản địa hóa cho mẫu tài liệu chuẩn.

Bản gốc tiếng Anh:

- [../../doc-template.md](../../doc-template.md)

## Cách dùng

- Dùng mẫu gốc làm chuẩn để tạo tài liệu mới.
- Giữ kỹ thuật ngữ, tên lệnh, khóa cấu hình ở tiếng Anh.
- Liên kết tài liệu mới vào `SUMMARY.md` và `docs-inventory.md`.
