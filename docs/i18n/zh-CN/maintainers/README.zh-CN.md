# 项目快照与分类文档

## 范围

项目快照是对开放 PR、Issue 和文档健康状况的有时间限制的评估。使用这些来：

- 识别功能开发导致的文档缺口
- 与代码变更一起优先安排文档维护
- 跟踪随时间变化的 PR/Issue 压力

对于稳定的文档分类（无时间限制），请使用 [docs-inventory.md](docs-inventory.zh-CN.md)。
