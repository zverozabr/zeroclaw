# Справочник команд (Русский)

Это первичная локализация Wave 1 для быстрого поиска CLI-команд ZeroClaw.

Оригинал на английском:

- [../../commands-reference.md](../../commands-reference.md)

## Когда использовать

- Поиск команд по типовой задаче
- Проверка флагов и границ поведения
- Сверка ожидаемого результата при отладке

## Правило

- Имена команд, флагов и ключей конфигурации сохраняются на английском.
- Финальная спецификация поведения — в английском оригинале.
